# CSS Variables

ZiggyUI uses CSS variables to allow you to customize the look and feel of the components. You can use the variables in your own CSS files or in your Svelte components. To make a change to the look and feel of a component, you can either override the CSS variables in your own CSS file or you can pass the variables as props to the component.
